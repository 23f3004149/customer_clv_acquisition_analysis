# Customer Lifetime Value vs Acquisition Cost — Seaborn Visualization

This repository contains a Seaborn-based scatterplot that visualizes the relationship between Customer Acquisition Cost (CAC) and Customer Lifetime Value (CLV). The analysis uses synthetic data to model realistic business behavior and demonstrates how analytics can support marketing optimization.

## Repository Contents
- chart.py — Script that generates synthetic data and produces a Seaborn scatterplot.
- chart.png — Final scatterplot image saved at exactly 512x512 pixel resolution.
- README.md — Documentation for the assignment.
